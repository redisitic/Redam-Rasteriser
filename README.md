# Redam-Rasteriser
A software rasterizer based on one made by Sebastian Lague (https://www.youtube.com/watch?v=yyJ-hdISgnw&t)

## Current Features:
- Perspective Projection
- Backface Culling
- Depth Buffering
- Animation Output
- Per-Triangle Parallelization
- Basic Directional and Ambient Lighting
